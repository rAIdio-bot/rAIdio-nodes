# rAIdio_nodes

ComfyUI custom nodes for rAIdio.bot.

## Nodes

**SaveAudioWAV** — Saves ComfyUI audio output as 16-bit PCM WAV. Simpler and more reliable than the built-in audio save node for downstream processing.
